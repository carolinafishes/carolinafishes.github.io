How To Interpret The Matrix

Needleman-Wunsch (Global): aligns full sequences end-to-end. The first row and first column are initialized using cumulative gap penalties (i * gap, j * gap), then every internal cell takes the max of: diagonal (match/mismatch), up (gap in sequence B), and left (gap in sequence A).

Smith-Waterman (Local): finds the best local subsequences. Boundaries start at 0 and every cell uses max(0, diagonal, up, left). Traceback starts at the highest scoring cell(s) and stops when score returns to 0.

Arrows in each cell show which predecessor(s) can lead to that score. Multiple arrows mean tied choices and potentially multiple optimal alignments.
